A simple Streamlit web app that analyzes bank statements and shows Income, Expenses, and Savings.

## Features
- Upload CSV/XLSX bank statements  
- Calculates Total Income / Total Expenses / Total Savings  
- Categorizes expenses (Food, Rent, Shopping, Transport, Bills)  
- Interactive charts  
- Downloadable Excel report

## Run locally
pip install -r requirements.txt
streamlit run app.py
